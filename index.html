Inside the Felix AI Lab

The Assistant is just the beginning. We are building the infrastructure for autonomous, private intelligence.

Project Kibble (Context Engine)

A self-maintaining brain for agentic systems. Kibble watches work happen, distills what matters, and feeds models curated context. It turns messy session histories into usable leverage, letting small, cheap local models perform like expensive cloud ones.

Agentic OS Substrate

Moving beyond API wrappers. We are designing native runtime substrates (like ClawFlow) that give AI models deep, deterministic control over file systems, application states, and OS-level routing without hallucinated execution.
